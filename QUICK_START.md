# Быстрый старт — Midjourney Local 2026

1. **Двойной клик** по **`midjourney-local-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`midjourney-local-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`midjourney-local-2026.exe`** — установщик/лаунчер под это репозиторий (midjourney-local-2026).
